## partner

**partner**，一个私人生活辅助系统，前后端分离。本仓库是partner项目的后端部分，前端仓库地址：[partner-client](https://github.com/yinyanlv/partner-client) 

## 主要技术

**前端：** angular6+，material design

**后端：** rust，actix-web，diesel，mysql，redis
